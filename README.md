# Superstore_sales_analysis
  Overview

Exploratory data analysis of a retail superstore dataset to evaluate sales performance, profitability, discounts, regions, and customer segments.

Tools

Python

Pandas

Matplotlib

Jupyter Notebook

Key Analysis

Top 5 products by total sales

Sales comparison by category

Discount vs Profit relationship

Sales by region

Sales by customer segment

Key Insights

The Consumer segment generates the highest sales.

Furniture is a leading revenue category.

Higher discounts are often associated with lower profits.

Sales performance varies across regions.

How to Run
pip install pandas matplotlib

Open the notebook and run all cells.
